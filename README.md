# TraceQ — BOQ Risk Review Engine

TraceQ analyses HVAC layout drawings (DXF) and automatically counts all equipment using a three-tier detection system:

1. **Layer Detection** — reads the drawing's layer names to classify equipment
2. **Block Detection** — matches block names against a known equipment dictionary
3. **Text Detection** — scans text labels and annotations for equipment references

Upload a drawing, get an instant equipment inventory with confidence scores.
